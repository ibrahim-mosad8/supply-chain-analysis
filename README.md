# supply-chain-analysis
#### Supply Chain Analysis Project
> Analysis of supply chain data to optimize operations and improve decision-making.

- Collected, cleaned, and analyzed supply chain data using **Excel** to track sales, pricing, and product performance.  
- Built **dynamic dashboards and reports** to monitor revenue, SKUs, and inventory-related metrics.  
- Generated **actionable insights** to support demand planning, inventory management, and operational efficiency.  
- Identified high-performing products and regions to guide strategic business decisions.
